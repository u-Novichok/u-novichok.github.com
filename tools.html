Kali

Auto-hides posts from a blocklist on X / Twitter

  • Chrome (v88+)
  • Only runs on x.com and twitter.com
  • Open source
Download Extension

How to install

  1. Unzip the downloaded file.
  2. Open Chrome and go to chrome://extensions
  3. Enable Developer mode using the toggle in the top-right.
  4. Click Load unpacked and select the Kali-main/extension folder.
  5. Done — the Kali icon will appear in your toolbar.
