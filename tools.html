Kali

Clean your X feed, Indian defence style

  • Chrome v88+
  • Edge
  • Brave
  • Only runs on x.com & twitter.com
  • Open source

Kali automatically hides unwanted posts on X (Twitter) using a community-managed blocklist — built for the Indian defence and OSINT community. Lightweight, no data collection, runs silently in the background.

Download v0.1.0

How to install

  1. Download and unzip kali-extension-v0.1.0.zip
  2. Open Chrome and go to chrome://extensions
  3. Enable Developer mode using the toggle in the top-right corner
  4. Click Load unpacked and select the extension folder from the unzipped files
  5. Done — the Kali icon will appear in your toolbar

Also works on Edge and Brave — same steps apply.
